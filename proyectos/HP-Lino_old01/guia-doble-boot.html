1) Requisitos y decisiones

Necesitas
  • Windows 10 oficial (Media Creation Tool).
  • Drivers HP (Elite 8300): chipset/MEI/red.
  • Focusrite Control + ASIO (Scarlett 2i2 3ª gen).
  • Instalador Cubase 10/12/14 + licencias.
  • MSCONFIG o EasyBCD (opcional) para el boot.
Decisiones
  • Windows 10 Pro 64‑bit (22H2) en el SSD nuevo.
  • Una licencia por instalación (en práctica suele reactivarse por huella en misma placa).
  • Separar particiones de cada Windows; proyectos en disco común.
Backup: haz copia de tus proyectos/ajustes antes de tocar el SSD viejo.

2) Crear USB e instalar Windows 10 Pro

  1. Crea USB booteable (≥8 GB) con Media Creation Tool.
  2. Desconecta físicamente el SSD viejo (con Cubase VST32).
  3. Conecta solo el SSD nuevo, arranca desde el USB.
  4. Instala Windows 10 Pro (sin Internet y con cuenta local). Elimina particiones del SSD nuevo y crea una nueva.
Atajos HP Elite 8300

Esc menú inicio · F9 elegir dispositivo · F10 BIOS Setup

Consejo: omite Cortana/OneDrive/telemetría en el OOBE si aparece.

3) Ajustes BIOS/UEFI (estabilidad)

Básicos
  • SATA Mode: AHCI.
  • Orden de arranque: SSD nuevo primero.
  • Desactiva PXE/Network Boot si no lo usas.
Rendimiento
  • Desactiva Spread Spectrum (si existe).
  • Activa Execute Disable Bit y VT‑x.
  • Ventiladores en Normal/Performance.

4) Drivers base

  1. Instala chipset/MEI y red (soporte HP Elite 8300).
  2. Conecta Internet y ejecuta Windows Update una sola vez para parches críticos.
  3. Opcional: drivers de almacenamiento extra si aplica.
Evita que Windows reemplace el driver de audio: bloquea actualizaciones de drivers en Configuración → Sistema → Información → Configuración avanzada del sistema → Hardware.

5) Scarlett 2i2 (3ª Gen) + ASIO

  1. Instala Focusrite Control (incluye ASIO). Reinicia si lo pide.
  2. Conecta la Scarlett a un puerto USB trasero (evita hubs). Desactiva ahorro de energía USB (siguiente sección).
  3. Define Sample Rate (44.1/48 kHz) y Buffer Size.
UsoBufferLatencia aprox.
Grabación128–256 samples6–10 ms
Mezcla512–1024 samples12–22 ms

6) Optimización de Windows para audio

Energía
  • Plan: Alto rendimiento.
  • Suspensión de disco: Nunca.
  • USB: desactiva Suspensión selectiva.
Audio
  • Sonido → usa Scarlett por defecto o desactiva audio integrado.
  • Propiedades → Avanzado: revisar control exclusivo.
  • Desactiva mejoras/efectos de Windows.
USB
  • Administrador de dispositivos → USB Root Hub → desmarca “Permitir que el equipo apague…”.
  • Evita hubs/puertos frontales no alimentados.
Procesos
  • Desactiva Game Bar/Game DVR y Modo Juego.
  • Windows Defender: añade exclusiones para proyectos y VST.
  • msconfig → inicio selectivo (opcional).

7) Instalar Cubase 10/12/14

  1. Instala Steinberg Download Assistant y descarga tu versión.
  2. Licencias: 10 (eLicenser), 12/14 (Steinberg Licensing).
  3. Rutas: VST2 C:\\VSTPlugins · VST3 C:\\Program Files\\Common Files\\VST3 · Librerías en disco de datos.
  4. Primer arranque: selecciona ASIO Focusrite y ajusta buffer según grabación/mezcla.

8) Configurar el Dual Boot

  1. Apaga y reconecta el SSD viejo.
  2. En BIOS (F10), deja el SSD nuevo primero.
  3. En Windows nuevo: abre msconfig → pestaña Arranque. Nombra entradas y ajusta tiempo (5–10 s). Si no aparece el antiguo, usa EasyBCD para añadirlo.
  4. Reinicia y prueba el menú.
Comandos útiles 
powercfg /h off
bcdedit /timeout 8
bcdedit /enum firmware
bcdedit /set {current} description "Windows Nuevo (Cubase 10/12/14)"
Licencia: normalmente se activa por huella de hardware al ser la misma placa; si no, activa con tu clave.

9) Proyectos y datos

Estrategia
  • Partición común D:\\Audio accesible por ambos Windows.
  • Estructura clara por DAW/cliente/fecha; evita espacios raros.
Compatibilidad
  • Congela/Exporta stems si migras de VST32.
  • Documenta plugins legacy en README.txt por proyecto.

10) Mantenimiento

  • Actualiza manualmente cada 2–3 meses (crea un Punto de restauración antes).
  • Exclusiones antivirus para carpetas de proyectos y VST.
  • Evita software innecesario en el sistema de audio.
  • Antes de sesiones: reinicia y cierra apps en segundo plano.
  • Haz una imagen del sistema cuando esté estable (Macrium Reflect).
