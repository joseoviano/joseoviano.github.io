Guía Windows 10 Pro USB Boot

Instalar Windows 10 Pro desde un USB

Guía práctica y directa: prepara un USB de arranque, configura BIOS/UEFI y realiza una instalación limpia sin sorpresas. Incluye tabla de configuración (MBR/GPT), resolución de errores y pasos posteriores.

Requisitos

Hardware mínimo
  • CPU 1 GHz o más rápida (x64 recomendado)
  • RAM: 4 GB (8 GB recomendado)
  • Almacenamiento: ≥ 32 GB (recomendado ≥ 128 GB SSD)
  • UEFI o BIOS, soporte de arranque por USB
Medios
  • USB 8 GB mínimo (16 GB mejor)
  • Imagen ISO de Windows 10 o Media Creation Tool
  • Utilidad alternativa: Rufus
Licencia
  • Clave de producto de Windows 10/11 Pro (25 caracteres)
  • Puedes omitirla y activar más tarde

Si vas a reutilizar un SSD/HDD, haz copia de seguridad. La instalación limpia puede borrar todas las particiones.

Crear el USB de arranque

Opción A — Herramienta oficial: Media Creation Tool (rápida y segura)
  1. En un PC con Windows, descarga Media Creation Tool para Windows 10 desde la web oficial de Microsoft.
  2. Conecta el USB (se borrará). Ejecuta la herramienta como administrador.
  3. Acepta licencia → Crear medios de instalación → Idioma y arquitectura (elige 64 bits).
  4. Selecciona Unidad flash USB → elige tu pendrive → Iniciar. La herramienta descargará y preparará el USB automáticamente.
  5. Al finalizar, expulsa el USB de forma segura.

Ventajas: gestiona particiones/UEFI automáticamente, descarga oficial, menos errores de arranque.

Opción B — Rufus + ISO (más control, GPT/MBR a medida)
  1. Descarga la ISO de Windows 10 (o usa opción “Descargar” de Rufus).
  2. Abre Rufus → Dispositivo: tu USB.
  3. Seleccionar: elige la ISO.
  4. Configura según tu firmware:
    Tu equipoEsquema de particiónDestino/ObjetivoSistema de archivos
    UEFI moderno (recomendado)GPTUEFI (no CSM)NTFS o FAT32*
    BIOS heredada / UEFI con CSMMBRBIOS o UEFI-CSMNTFS

    *Si el archivo install.wim supera 4 GB, FAT32 no lo admite. Rufus usa NTFS con UEFI:NTFS para seguir arrancando en UEFI.

  5. Etiqueta de volumen opcional → pulsa Empezar. Acepta el formateo.
  6. Al terminar, expulsa el USB de forma segura.
Cómo saber si mi equipo usa UEFI o BIOS
  • Windows actual: Win+R → msinfo32 → Modo de BIOS: UEFI o Heredado.
  • En UEFI, puedes tener Secure Boot activado. No es obligatorio desactivarlo para Windows 10.

Configurar BIOS/UEFI para arrancar desde USB

  1. Con el equipo apagado, conecta el USB.
  2. Enciende y pulsa repetidamente la tecla de arranque/BIOS: suele ser F12, F8, F11 (menú de arranque) o Del/Supr, F2 (BIOS).
  3. En el Boot Menu, elige tu USB. Si no aparece:
    • En BIOS/UEFI → pestaña Boot: mueve el USB a la primera posición.
    • En UEFI: comprueba que el modo coincide con cómo creaste el USB (UEFI vs CSM).
    • Desactiva temporalmente Fast Boot si oculta dispositivos.

En muchos portátiles: F12 (Dell/Lenovo), Esc→F9 (HP), F8/F12 (Acer), F2 BIOS en ASUS/MSI.

Instalación de Windows 10 Pro (limpia)

  1. Arranca desde el USB. En el instalador, elige idioma, formato de hora y teclado → Siguiente → Instalar ahora.
  2. Introduce la clave de producto o pulsa No tengo clave. Selecciona Windows 10 Pro.
  3. Acepta licencia → Personalizada: instalar solo Windows (avanzado).
  4. En ¿Dónde quieres instalar Windows?:
    • Para instalación limpia: elimina las particiones del disco de destino (cuidado) hasta que quede Espacio sin asignar.
    • Selecciona el espacio sin asignar y pulsa Siguiente. El asistente creará particiones necesarias (MSR/EFI).
    • Si instalas junto a otro SO, elige una partición con tamaño suficiente y pulsa Formatear.
  5. Espera la copia de archivos y reinicios. No pulses ninguna tecla cuando reinicie.
  6. Configura la experiencia inicial (región, teclado, red, cuenta, privacidad). Puedes crear cuenta local si lo prefieres.

Consejo: si usas SSD NVMe y el disco no aparece, entra en BIOS y pon modo AHCI y actualiza BIOS si es muy antigua.

Después de instalar

Drivers y Windows Update
  • Conéctate a Internet → Configuración > Windows Update (instala todo).
  • Descarga controladores del fabricante (chipset, vídeo, audio, Wi‑Fi). Reinicia.
Activación
  • Configuración > Actualización y seguridad > Activación.
  • Si tu placa tenía licencia digital, se activará automáticamente al conectarte.
Privacidad y rendimiento
  • Desactiva lo que no uses en Privacidad.
  • Habilita Restaurar sistema y crea un punto de restauración.
  • Opcional: crea imagen del sistema con Recuperación o herramienta de terceros.
Software básico
  • Navegador, gestor de archivos comprimidos, visor PDF, suite ofimática.
  • Habilita Protección contra ransomware en Seguridad de Windows.

Errores comunes y solución

El USB no arranca o no aparece en el menú
  • Recrea el USB con el esquema correcto (GPT/UEFI o MBR/BIOS).
  • Usa otro puerto USB (mejor 2.0 en equipos antiguos).
  • Desactiva Fast Boot y habilita USB boot en BIOS.
“No se pudo crear una partición nueva” / “No se puede instalar en este disco”
  • Si tu firmware es UEFI, el disco debe estar en GPT. En la pantalla de particiones, usa Shift+F10 → diskpart → list disk → select disk X → clean → convert gpt (borra todo).
  • Para BIOS heredada, usa MBR: convert mbr.
Falta un controlador de medios
  • Conecta el USB a un puerto 2.0 (controladores 3.x viejos dan problemas).
  • Vuelve a crear el USB; evita hubs/cables defectuosos.
Bucle de reinicio tras copiar archivos
  • Al primer reinicio, no arranques de nuevo desde el USB. Quita prioridad o retíralo.
Clave de producto rechazada
  • Completa la instalación sin clave y activa más tarde con Cambiar la clave.

Atajos útiles

  • Shift+F10 abre consola (DiskPart) durante la instalación.
  • Alt+F4 cierra ventanas emergentes del instalador.
  • Captura de pantalla: Win+PrtSc (ya en sistema instalado).
