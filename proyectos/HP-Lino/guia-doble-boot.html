Guía completa: Doble boot de Windows 10 para Cubase en HP Elite 8300 (32 GB RAM) + Scarlett 2i2

Objetivo: dejar un SSD nuevo con Windows 10 Pro 22H2 optimizado para audio (Cubase 10/12/14) y mantener otro SSD con Windows 10 + Cubase VST32 para proyectos antiguos, con un menú de arranque para elegir en el inicio.

Equipo HP Elite 8300 · RAM 32 GB · Interfaz Focusrite Scarlett 2i2 3ª Gen · DAW Cubase 10/12/14 y Cubase VST32

1) Requisitos y decisiones previas

Software necesario
  • Imagen oficial de Windows 10 (Media Creation Tool).
  • Drivers HP (chipset/MEI/Intel RST si aplica) para Elite 8300.
  • Focusrite Control + Driver ASIO.
  • Instalador de Cubase (10/12/14) + Steinberg Download Assistant / eLicenser o Steinberg Licensing.
  • Herramienta de arranque: MSCONFIG o EasyBCD (opcional, facilita).
Decisiones clave
  • Windows 10 Pro 64 bit (22H2) para el SSD nuevo.
  • Licencia: idealmente, una por instalación. En la práctica, en misma placa suele reactivar por hardware; si no, activa más tarde.
  • Cada Windows con su propia partición (no mezclar Usuarios/Documentos entre sistemas).
  • Proyectos en una carpeta común de datos (tercer disco/partición compartida).
Backup: si el SSD viejo contiene proyectos/ajustes críticos, realiza copia de seguridad antes de tocar nada.

2) Crear USB de instalación y preparar el SSD nuevo

  1. En otro PC o en el mismo, descarga el Media Creation Tool y crea un USB booteable (8 GB o más).
  2. Desconecta físicamente el SSD viejo (el que tiene el Windows con Cubase VST32) para proteger su arranque.
  3. Conecta solo el SSD nuevo. Inserta el USB y enciende el PC.
Acceder al menú de arranque/BIOS en HP Elite 8300
EscF9F10
  • Esc al encender → menú de inicio.
  • F9 → seleccionar dispositivo de arranque (USB).
  • F10 → BIOS Setup.

3) Instalar Windows 10 Pro (SSD nuevo)

  1. Arranca desde el USB de Windows 10. Elige Edición: Pro, idioma y teclado.
  2. Cuando pregunte por clave, pulsa No tengo clave. (Activaremos después).
  3. En Opciones de unidad: elimina particiones del SSD nuevo y crea una nueva (Windows generará las necesarias). Formatea si es preciso.
  4. Completa la instalación sin conectar a Internet para evitar drivers genéricos automáticos.
  5. Crea una cuenta local (nombre sencillo, sin espacios para ruta corta; p. ej. Jose).
Consejo: omite Cortana, OneDrive y telemetría durante el OOBE si se ofrece. Lo activaremos solo si lo necesitas.

4) Ajustes BIOS/UEFI recomendados (HP Elite 8300)

Básicos
  • SATA Mode: AHCI (no IDE). Si cambias desde IDE en un Windows ya instalado, habilita AHCI en registro antes.
  • Boot Order: pon SSD nuevo como primero.
  • Desactiva Network/PXE Boot si no lo usas.
Rendimiento y estabilidad
  • Desactiva Spread Spectrum (si está disponible) para estabilidad de reloj.
  • Activa Execute Disable Bit / VT-x (no afecta audio, pero útil para herramientas).
  • Fija ventiladores en perfil Normal o Performance para evitar thermal throttling.

5) Instalar drivers base y red

  1. Instala **chipset/MEI** y controladores de red desde la página de soporte de HP para Elite 8300.
  2. Conecta Internet y ejecuta Windows Update una vez para parches críticos (no dejes instalación automática activa).
  3. Opcional: si usas almacenamiento adicional SATA/NVMe con drivers, instálalos ahora.
Evita que Windows reemplace más tarde el driver de audio por uno genérico: bloquea actualizaciones de drivers en Configuración → Sistema → Información → Configuración avanzada del sistema → Hardware → Configuración de instalación de dispositivos.

6) Instalar Focusrite Scarlett 2i2 (3ª Gen) y ASIO

  1. Descarga e instala Focusrite Control (incluye el driver ASIO). Reinicia si lo pide.
  2. Conecta la Scarlett directamente a un puerto USB trasero (evita hubs). Desactiva ahorro de energía del USB (ver sección 7).
  3. En Focusrite Control, define Sample Rate (44.1/48 kHz) y revisa Buffer Size.
UsoBuffer sugeridoLatencia aproximada
Grabación (monitoreo en vivo)128–256 samples6–10 ms
Mezcla (muchos plugins)512–1024 samples12–22 ms

7) Optimización de Windows para audio (baja latencia)

Plan de energía
  • Panel de control → Opciones de energía → Alto rendimiento.
  • Suspensión de disco: Nunca. Estado mínimo del procesador: 100% (opcional si notas picos de DPC).
  • USB: desactiva Suspensión selectiva.
Audio y dispositivos
  • Panel de control → Sonido → pestaña Reproducción/Grabación: establece la Scarlett como predeterminada o deshabilita audio integrado.
  • Propiedades de los dispositivos → pestaña Avanzado: desmarca Permitir que las aplicaciones tomen el control exclusivo (según flujo de trabajo).
  • Desactiva Mejoras/efectos.
USB y energía
  • Administrador de dispositivos → Controladoras de bus serie universal → cada Hub / Root Hub → Propiedades → Administración de energía → desmarca Permitir que el equipo apague…
  • Evita puertos frontales/hubs no alimentados.
Servicios y procesos
  • Desactiva Game Bar / Game DVR y Modo Juego.
  • Desinstala bloatware/OneDrive si no lo usas.
  • Antivirus: usa Windows Defender con exclusiones para carpetas de proyectos y VST.
Opcional avanzado (con punto de restauración)
  • Deshabilitar HPET o limitar Timers solo si miden DPC alto y comprueban mejora.
  • En msconfig → Inicio selectivo: deshabilita programas no críticos.

8) Instalar Cubase 10/12/14

  1. Instala Steinberg Download Assistant. Descarga tu versión (10/12/14) y contenido.
  2. Licencias:
    • Cubase 10: eLicenser USB/Soft-eLicenser.
    • Cubase 12/14: Steinberg Licensing con cuenta (sin dongle).
  3. Ubicaciones sugeridas:
    • VST2 (si procede): C:\\VSTPlugins
    • VST3: C:\\Program Files\\Common Files\\VST3 (por defecto)
    • Bibliotecas pesadas: en un disco de datos dedicado (no en C:).
  4. Primer arranque: selecciona driver ASIO Focusrite y ajusta buffer según tarea.
Tip: crea plantillas (Templates) de proyecto para Grabación y Mezcla con buffers distintos. Ahorra tiempo.

9) Configurar el arranque dual (Dual Boot)

  1. Apaga el PC y reconecta el SSD viejo (Windows + Cubase VST32).
  2. En BIOS (F10), asegura que el SSD nuevo es el primer disco de arranque.
  3. Arranca en Windows nuevo. Abre msconfig → pestaña Arranque.
    • Si detecta ambas instalaciones, nómbralas: Windows Nuevo (Cubase 10/12/14) y Windows Antiguo (VST32).
    • Si no aparece el antiguo, usa EasyBCD → Add New Entry → Windows → selecciona la partición del SSD viejo → Add.
    • Establece el sistema por defecto y el tiempo de espera (p. ej. 5–10 s).
  4. Reinicia y comprueba el menú de selección al inicio.
Comandos útiles (avanzado) 
powercfg /h off
bcdedit /timeout 8
bcdedit /enum firmware
bcdedit /set {current} description "Windows Nuevo (Cubase 10/12/14)"
Licencia de Windows: lo habitual es que se active por huella de hardware al ser la misma placa. Si no, activa manualmente con tu clave o compra una adicional.

10) Estructura de proyectos y datos compartidos

Estrategia recomendada
  • Crea una partición/disco D:\\Audio accesible desde ambos Windows.
  • Estructura por DAW/cliente/fecha: D:\\Audio\\Cubase14\\<Proyecto>, ...\\CubaseVST32\\<Proyecto>.
  • Evita rutas con espacios/caracteres raros.
Compatibilidad proyectos antiguos
  • Congela/Exporta stems de proyectos VST32 si los migras a versiones nuevas.
  • Documenta plugins legacy usados (versión exacta) en un README.txt por proyecto.

11) Mantenimiento y buenas prácticas

  • Desactiva Windows Update automático; actualiza manualmente cada 2–3 meses y antes crea un Punto de restauración.
  • Exclusiones en antivirus para carpetas de proyectos y VST.
  • Evita instalar software innecesario en el sistema de audio.
  • Antes de sesiones importantes: reinicia el PC, desconecta apps en segundo plano (navegadores, mensajería).
  • Haz imagen del sistema con Macrium Reflect o similar una vez esté estable.
