Requisitos de software (proceso completo)

Windows y controladores

  • Windows 10 Pro 64‑bit (22H2): ISO o Media Creation Tool.
  • Drivers HP Elite 8300: Chipset, Intel MEI, LAN/Wi‑Fi, (RST si aplica).
  • Actualizaciones críticas: Windows Update (una pasada inicial).

Audio y DAW

  • Focusrite Control + ASIO (Scarlett 2i2 3ª Gen).
  • Steinberg Download Assistant (instaladores Cubase).
  • Licencias:
    • Cubase 10/10.5: eLicenser (USB).
    • Cubase 12/14: Steinberg Licensing (sin dongle).

Utilidades recomendadas

  • EasyBCD (opcional): gestionar entradas de arranque.
  • Macrium Reflect (opcional): imagen/backup del sistema estable.
  • LatencyMon (opcional): diagnóstico de latencia DPC.

Estructura de datos

  • Carpeta común de proyectos: D:\Audio.
  • VST2: C:\VSTPlugins · VST3: C:\Program Files\Common Files\VST3.
  • Bibliotecas pesadas en disco/partición de datos (no en C:).

Guía: Doble boot Windows 10 para Cubase

Equipo HP Elite 8300 · 32 GB · SSD · Scarlett 2i2 (3ª Gen) · Windows 10 Pro 22H2

1) Requisitos y decisiones

Necesitas
  • Windows 10 (Media Creation Tool).
  • Drivers HP Elite 8300 (chipset/MEI/red).
  • Focusrite Control + ASIO.
  • Instaladores de Cubase 10/12/14 + licencias.
Decisiones
  • W10 Pro 64‑bit (22H2) en SSD nuevo.
  • Licencia por instalación (suele reactivarse por huella en misma placa).
  • Particiones separadas; proyectos en disco común.
Backup Copia tus proyectos/ajustes antes de tocar el SSD viejo.

2) Crear USB e instalar

  1. USB booteable (≥8 GB) con Media Creation Tool.
  2. Desconecta el SSD viejo (VST32).
  3. Conecta solo el SSD nuevo y arranca desde el USB.
  4. Instala Windows 10 Pro (sin Internet, cuenta local). Borra y crea partición nueva.

Esc menú inicio · F9 elegir dispositivo · F10 BIOS (HP Elite 8300)

Omitir Cortana/OneDrive/telemetría en OOBE.

3) Ajustes BIOS/UEFI

Básicos
  • SATA: AHCI
  • Boot: SSD nuevo primero
  • Desactivar PXE/Network Boot
Rendimiento
  • Desactivar Spread Spectrum (si existe)
  • Activar Execute Disable Bit y VT‑x
  • Ventiladores Normal/Performance

4) Drivers base

  1. Instala chipset/MEI y red (HP).
  2. Una pasada de Windows Update (parches críticos).
  3. Opcional: controladores de almacenamiento adicionales.
Evita que Windows reemplace el driver de audio: bloquea actualizaciones de drivers (Configuración avanzada del sistema → Hardware).

5) Scarlett 2i2 + ASIO

  1. Instala Focusrite Control (ASIO). Reinicia si lo pide.
  2. Conecta a USB trasero (evita hubs). Desactiva ahorro de energía USB.
  3. Define Sample Rate (44.1/48) y Buffer.
UsoBufferLatencia aprox.
Grabación128–2566–10 ms
Mezcla512–102412–22 ms

6) Optimización Windows

Energía
  • Alto rendimiento
  • Disco: Nunca apagar
  • USB: sin suspensión selectiva
Audio
  • Scarlett por defecto o desactivar integrado
  • Control exclusivo (revisar)
  • Sin mejoras/efectos
USB
  • USB Root Hub → desmarcar “Permitir que apague…”
  • Evitar hubs/puertos frontales
Procesos
  • Sin Game Bar/Game DVR/Modo Juego
  • Defender: exclusiones Proyectos/VST
  • msconfig → inicio selectivo (opcional)

7) Instalar Cubase 10/12/14

  1. Instala Steinberg Download Assistant.
  2. Licencias: 10 (eLicenser), 12/14 (Steinberg Licensing).
  3. Rutas: VST2 C:\VSTPlugins · VST3 C:\Program Files\Common Files\VST3 · Librerías en disco de datos.
  4. Primer arranque: ASIO Focusrite y buffer adecuado.

8) Configurar el Dual Boot

  1. Apaga y reconecta el SSD viejo.
  2. En BIOS (F10), SSD nuevo primero.
  3. En Windows nuevo: msconfig → Arranque (nombres y timeout 5–10 s). Si no aparece, usar EasyBCD.
  4. Reinicia y prueba.
powercfg /h off
bcdedit /timeout 8
bcdedit /enum firmware
bcdedit /set {current} description "Windows Nuevo (Cubase 10/12/14)"
Licencia: suele activarse por huella; si no, activa con tu clave.

9) Proyectos y datos

Estrategia
  • Carpeta común D:\Audio
  • Estructura por DAW/cliente/fecha
Compatibilidad
  • Congelar/exportar stems desde VST32
  • Documentar plugins legacy en README.txt

10) Mantenimiento

  • Actualizar manualmente cada 2–3 meses (con Punto de restauración).
  • Exclusiones antivirus para Proyectos y VST.
  • Sistema “limpio” en el Windows de audio.
  • Reiniciar antes de sesión; cerrar apps en segundo plano.
  • Imagen del sistema estable (Macrium Reflect).

Optimización de Windows 10 para Grabación de Audio

Objetivo Latencia baja y estabilidad ASIO con Scarlett 2i2 + Cubase.

1) Energía y rendimiento

  • Opciones de energía → Alto rendimiento.
  • Disco: Nunca apagar.
  • USB: Desactivar suspensión selectiva.
  • CPU mínimo = 100% (si hay picos DPC, baja a 5–10%).
  • BIOS (opcional): desactivar C‑States si hay inestabilidad.

2) Drivers y dispositivos

  • Instalar Focusrite Control (ASIO oficial).
  • Sonido → deshabilitar dispositivos no usados.
  • Actualizar chipset/MEI/red (HP Elite 8300).

3) Latencia y buffers

  • Focusrite: Grabación 128–256 · Mezcla 512–1024.
  • Cubase: usar ASIO Focusrite.
  • Monitoreo directo (Direct Monitor) si necesitas casi cero latencia.

4) Servicios y procesos

  • Windows Update: Pausar durante sesiones.
  • Considera desactivar temporalmente Windows Search si indexa intensamente.
  • Antivirus: exclusiones para Proyectos y VST.

5) Efectos y notificaciones

  • Sonidos de Windows: Sin sonidos.
  • Asistente de concentración: activado.
  • Efectos visuales: desactivar animaciones/transparencias.

6) Disco y memoria

  • SSD con ≥20% libre.
  • Optimizar unidades (TRIM) para SSD; desfragmentar solo HDD.
  • Librerías pesadas en disco/partición de datos.
